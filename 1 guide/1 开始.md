# HotPush - 热点聚合推送平台

## 项目背景

HotPush 是一个热点聚合推送平台，旨在帮助用户实时追踪各大平台的热点内容，并通过多种渠道进行推送通知。平台支持自定义抓取规则、定时任务调度、AI 摘要生成等功能，帮助用户高效获取有价值的信息。

## 核心功能

1. **多源热点聚合**：支持微博、知乎、B站、V2EX、Hacker News 等多个平台的热点抓取
2. **灵活推送渠道**：支持 Telegram、Discord、邮件、企业微信、飞书、钉钉等推送方式
3. **智能抓取规则**：可配置抓取间隔、超时时间、重试次数等参数
4. **AI 摘要生成**：集成 LiteLLM，支持 OpenAI、Claude、DeepSeek、Ollama 等多种 AI 模型
5. **用户认证管理**：基于 JWT 的用户认证系统
6. **定时任务调度**：使用 APScheduler 实现定时抓取和推送
7. **数据缓存优化**：Redis 缓存热点数据，提升访问性能
8. **响应式前端界面**：Vue 3 + Tailwind CSS 构建的现代化管理界面

## 技术架构

### 后端技术栈

- **Web 框架**：FastAPI (Python 3.10+)
- **数据库**：MySQL 8.0+
- **缓存**：Redis (可选)
- **任务调度**：APScheduler
- **RSS 解析**：feedparser
- **HTTP 客户端**：httpx
- **AI 集成**：LiteLLM
- **认证**：JWT + bcrypt

### 前端技术栈

- **前端框架**：Vue 3
- **构建工具**：Vite 5
- **UI 框架**：Tailwind CSS 3.4
- **状态管理**：Pinia
- **路由**：Vue Router 4
- **图表**：Chart.js + vue-chartjs

### 部署架构

- **容器化**：Docker + Docker Compose
- **反向代理**：Nginx (前端)
- **进程管理**：Uvicorn (后端)

## 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/sail-djl/hotpush.git
   cd hotpush
   ```

2. **环境配置**
   ```bash
   # 后端环境变量
   cp backend/.env.example backend/.env
   # 编辑 .env 文件，配置数据库等信息
   ```

3. **启动开发环境**
   ```bash
   # 使用 Docker Compose (推荐)
   docker-compose up -d
   
   # 或手动启动
   # 后端
   cd backend
   pip install -r requirements.txt
   uvicorn app.main:app --reload
   
   # 前端
   cd frontend
   npm install
   npm run dev
   ```

4. **访问应用**
   - 前端界面：http://localhost:5173
   - API 文档：http://localhost:8000/docs
   - 管理后台：http://localhost:5173/login

## 常见问题

### Q: 如何添加新的热点源？

A: 在 `backend/app/utils/sources.py` 中添加新的源配置，然后在前端 `frontend/src/utils/sourceIcons.js` 中添加对应图标。

### Q: 如何配置新的推送渠道？

A: 在 `backend/app/config.py` 中添加新的配置项，然后在 `backend/app/services/push_service.py` 中实现推送逻辑。

### Q: 如何自定义 AI 模型？

A: 在 `.env` 文件中配置 `AI_MODEL`、`AI_API_KEY`、`AI_BASE_URL` 等环境变量，支持 OpenAI、Claude、DeepSeek、Ollama 等模型。
