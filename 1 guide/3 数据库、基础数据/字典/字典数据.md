## 推送规则字典

### 规则类型

| 规则类型 | 类型标识 | 配置参数 | 说明 |
|----------|----------|----------|------|
| 关键词规则 | keyword | keywords, match_type | 根据关键词过滤热点 |
| 数据源规则 | source | sources | 指定推送的数据源 |
| 排名规则 | rank | min_rank, max_rank | 根据排名阈值过滤 |

### 规则配置示例

#### 关键词规则

```json
{
  "keywords": ["Python", "AI", "机器学习"],
  "match_type": "any"
}
```

- `match_type`：匹配类型
  - `any`：匹配任意关键词
  - `all`：匹配所有关键词
  - `exact`：精确匹配

#### 数据源规则

```json
{
  "sources": ["weibo", "zhihu", "bilibili"]
}
```

#### 排名规则

```json
{
  "min_rank": 1,
  "max_rank": 10
}
```

## 错误码字典

### API 错误码

| 错误码 | 说明 | 处理建议 |
|--------|------|----------|
| 400 | 请求参数错误 | 检查请求参数 |
| 401 | 未授权 | 检查 Token 或重新登录 |
| 403 | 权限不足 | 检查用户权限 |
| 404 | 资源不存在 | 检查请求路径 |
| 422 | 数据验证失败 | 检查数据格式 |
| 500 | 服务器内部错误 | 查看服务器日志 |
| 502 | 网关错误 | 检查服务状态 |
| 503 | 服务不可用 | 稍后重试 |

### 业务错误码

| 错误码 | 说明 | 处理建议 |
|--------|------|----------|
| 1001 | 数据源不存在 | 检查数据源配置 |
| 1002 | 数据源已存在 | 使用其他源标识 |
| 1003 | 推送渠道不存在 | 检查渠道配置 |
| 1004 | 推送渠道已存在 | 使用其他渠道标识 |
| 1005 | 推送规则不存在 | 检查规则配置 |
| 1006 | 用户已存在 | 使用其他用户名 |
| 1007 | 用户名或密码错误 | 检查登录信息 |
| 1008 | Token 已过期 | 重新登录 |

## 日志级别字典

### 日志级别

| 级别 | 说明 | 使用场景 |
|------|------|----------|
| DEBUG | 调试信息 | 开发调试 |
| INFO | 普通信息 | 正常运行日志 |
| WARNING | 警告信息 | 潜在问题 |
| ERROR | 错误信息 | 错误但可继续运行 |
| CRITICAL | 严重错误 | 严重错误，可能崩溃 |

### 日志格式

```
2026-07-06 12:00:00 [INFO] app.main: 应用启动成功
2026-07-06 12:00:01 [ERROR] app.services.push_service: 推送失败: Connection timeout
```

## 时间格式字典

### 时间格式

| 格式 | 说明 | 示例 |
|------|------|------|
| %Y-%m-%d | 日期 | 2026-07-06 |
| %H:%M:%S | 时间 | 12:00:00 |
| %Y-%m-%d %H:%M:%S | 日期时间 | 2026-07-06 12:00:00 |
| %Y-%m-%dT%H:%M:%S | ISO 格式 | 2026-07-06T12:00:00 |
| %Y-%m-%dT%H:%M:%SZ | UTC 格式 | 2026-07-06T12:00:00Z |

### 时区配置

- 默认时区：Asia/Shanghai
- UTC 偏移：+08:00
- 数据库存储：UTC 时间
- 前端显示：本地时间
